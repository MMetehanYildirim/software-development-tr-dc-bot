# software-development-tr-dc-bot
Software Development tr topluluğu için geliştirilen bir dc botudur. 

#### Teknolojiler
* [Node.js](https://nodejs.org/en/download/)
* [Discord.js](https://discord.js.org/#/)
* [Moment.js](https://momentjs.com)

#### Yükleme
Öncelikle buradan [Node.js Website](https://nodejs.org/en/download/) Node.js indirip kurmanız gereklidir.
Editör olarakta Vs Code kullanabilirsiniz.

Node.js yükledikten sonra projeyi klonlayabilir veya bilgisayarınıza indirebilirsiniz.

Klon:

> cd projeyolu

> git clone https://github.com/pars11/software-development-tr-dc-bot

Bot tokenini "bot.login('');" alanına app.js dosyasına eklemeniz gereklidir.

Discord app ve bot token hakkında detaylı bilgiye buradan ulaşabilirsiniz. [Detay](https://github.com/moonstar-x/discord-downtime-notifier/wiki/Getting-a-Discord-Bot-Token)

Tokeninizi girdikten sonra bot hazır olacaktır. Aşağıdaki kodu cmd ekranınıza yazarak botu çalıştırabilirsiniz.

> node app.js
